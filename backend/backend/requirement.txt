Instructions to run the project locally : 

1.Clone the repository : 
     git clone <repository_url>
cd Appointment-Booking-System


2. Install Python dependencies: Navigate to the backend folder and install the required Python packages:
    cd backend
pip install -r requirements.txt

3. flask
flask-cors

4. In the backdend folder run the flask server : 
     python app.py


5. To run the frontend right click on index.html ther you will find an option "Open with live server" , click on that and your frontend will be loaded.
